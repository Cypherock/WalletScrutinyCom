Here we test if version `3.3.0` is verifiable after Blockstream provided the
missing tag and some more build instructions. See older reviews for more.

The extra build instructions we are trying now are:

> You can manually run `./app/fetch_gdk_binaries.sh` script to update gdk binaries from [Blockstream/gdk](https://github.com/Blockstream/gdk): green release_3.3.0 uses gdk v0.0.24 and the released gdk binaries are available at here, for android package `gdk-android-jni.tar.gz`. (Documentation should be improved)

```
$ cd green_android
$ git pull
$ git checkout $( git tag | grep 3.3.0 )
$ docker run -it --volume $PWD:/mnt --workdir /mnt --rm mycelium-wallet bash
```

... and this is what we want to see. Only files from the signature and from our
own unpacking of APKs differ.

The last two not verifiable verdicts were due to lacking build instructions and
we hope the provider updates its documentation accordingly.

Our verdict: This wallet is **verifiable**.
